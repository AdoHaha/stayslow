# stayslow

A small Chrome Manifest V3 extension that adds a mandatory countdown before opening selected websites.

Default slowed domains:

- `youtube.com`
- `reddit.com`

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this directory.

Use the extension options page to edit domains, notes, the delay length, and the master enabled toggle.
The countdown page has no bypass button; it only shows a few useful alternative links while waiting.
Clicking the extension icon opens that options page in a normal browser tab.
